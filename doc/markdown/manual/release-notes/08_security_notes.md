# Security Notes

Starting with OCS and GCS v9.1.0, several security enhancements have been introduced to strengthen the security posture of the cluster scheduler components:

-**MUNGE Authentication Support** – Enables secure authentication of users and processes interacting with the cluster scheduler.
-**TLS Encryption** – Supports TLS-secured communication between OCS, GCS, and other scheduler components to protect data in transit.

These features rely on packages provided by the supported operating systems. Ensure that your system is properly configured and up to date, and follow security best practices to fully benefit from these enhancements.
